# Simple Blog Web App

A simple blog application built with React 19, TypeScript, Redux Toolkit, and Supabase.

---

## Tech Stack

- React 19
- TypeScript
- Redux Toolkit
- React Router
- Supabase (Auth + Database)

---

## Features

Authentication:
- Register
- Login
- Logout
- Email confirmation disabled

Blog Management:
- Create blog
- Update blog
- Delete blog
- List blogs
- Pagination

Security:
- Row Level Security (Supabase)
- Only owners can edit/delete their blogs
- Protected routes
